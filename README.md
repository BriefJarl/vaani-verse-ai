# 🎙️ Vaani Verse — AI Study Assistant

Voice-first AI system that plans, reschedules, and validates learning through quizzes.  
Combines multi-agent architecture (Planner, Tutor, Analytics) with real-time study optimization.  
Ensures tasks are completed only after proven understanding, improving retention.  
Built with Node.js, React Native, AWS, and Supabase for scalable performance.
